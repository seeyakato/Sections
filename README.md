# Sections  

HTML タグ集  
======================  
  
HTMLのタグが書かれています。  
  
  
使い方
------
このテンプレートをpushする手順の例です。

このリポジトリをクローンします。

    $ git clone https://github.com/seeyakato/Sections.git

クローンしたリポジトリの中へ移動します。

    $ cd Sections  
  
編集します  

`README.md` を新しいプロジェクトに合わせて編集します。

    $ $EDITOR README.md

コミットします。

    $ git commit -m "HTML" 

プッシュします。

    $ git push
